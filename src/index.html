Foxy the Fox

As a solo developer, I brought this captivating 2D platformer to life, featuring the clever fox, Foxy, on a journey through lush forests, treacherous caves, and mysterious ruins. I handled every aspect, from character design and pixel art to coding Foxy's controls and crafting engaging levels.

With Windows Forms, I designed the game's graphical user interface, carefully placing windows, buttons, and UI elements to enhance the player's experience. Beneath the surface, I implemented complex logic to handle Foxy's movements, physics simulations for gravity and velocity, and meticulous collision detection to ensure a smooth and immersive gameplay. Foxy's animations and behaviors are all orchestrated through event-driven programming and game loops.

Personal projects

Bash scripting

CPU.SH

is my go-to script for monitoring and managing CPU-related tasks on my Linux machine. It displays CPU information, usage statistics, and even allows me to set performance profiles. It's handy for optimizing my system's performance, especially during resource-intensive tasks.

DISK.SH

With disk.sh, I keep an eye on my disk usage and perform maintenance tasks as needed. It provides me with a snapshot of my disk space and helps me identify areas where I might need to clean up or optimize storage. Keeping my disks tidy is essential for efficient development.

PROJ.SH

projectsD.sh is a script that simplifies the management of my personal development projects. It might automate tasks like creating project directories, initializing Git repositories, or setting up project-specific environments. This script streamlines my workflow and ensures consistency across my projects.
